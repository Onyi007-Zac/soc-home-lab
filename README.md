# SOC Home Lab

## Objective
To build a Security Operations Center (SOC) home lab for detecting and analyzing
security events using a SIEM.

## Architecture
- Windows 10 Victim Machine
- Attacker Machine (Kali Linux)
- Wazuh SIEM
- Sysmon for detailed logging

## Tools Used
- Wazuh
- Sysmon
- Windows Event Viewer
- Kali Linux

##  Use Cases
- Brute force attack detection
- Suspicious process creation
- Unauthorized login attempts

## Expected Outcome
- Logs ingested into SIEM
- Alerts generated
- Incident reports created
